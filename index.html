freeCodeCamp project
25+5 Clock
25+5 Hours is a timer that consists of two parts: a break and a session. You can adjust a session time and a break time, then press "Start", when the session time is over, the break time will start and it will repeat until you press "Pause" or "Reset". "Reset" returns to default settings. This is a small project that was made as one of the tasks of the freeCodeCamp certificate. I am using knowledge of React to do this.
What I used: HTML, CSS, React JS.
